# Python DevOps Automation Projects

This repository contains **real-world Python automation scripts** built while learning **Python for DevOps and Cloud Engineering**.

Instead of focusing only on Python syntax, these projects solve practical infrastructure and automation problems that engineers face in production environments.

The goal of this repository is to build strong Python scripting skills required for differnt roles in Cloud & DevOps feild

## Projects Included

### Script1. Server Health Monitoring Script

Monitors system resources continuously.

Features:

* CPU monitoring
* RAM monitoring
* Disk usage monitoring
* Warning threshold detection
* CSV report generation
* Log file generation

---

### Script2. Website Monitoring Script

Checks whether a website is up or down.

Features:

* URL monitoring
* Status code checking
* Response time measurement
* Website availability detection
* CSV reporting
* Logging

---

### Script3. Log File Analyzer

Reads server logs and analyzes log severity.

Features:

* Count ERROR logs
* Count WARNING logs
* Count INFO logs
* Generate log reports
* Save results in CSV
* Logging support

---

### Script4. Disk Cleanup Automation Script

Automates cleanup of unnecessary files.

Features:

* Scan directory
* Detect file extensions
* Delete log files
* Count deleted files
* Cleanup logging

---

### Script5. Service Health Checker

Checks if important services are running.

Features:

* Process detection
* Service status monitoring
* Detect running/not running services
* Count healthy services
* Generate reports

Examples:

* Docker
* Nginx
* Jenkins

---

### Script6. Backup Automation Script

Creates compressed backup of folders.

Features:

* Directory backup
* ZIP file creation
* Backup size calculation
* Timestamp-based backup naming
* Backup logging

---

### Script7. Linux Command Runner + Troubleshooter

Runs Linux troubleshooting commands automatically.

Features:

* Execute Linux commands
* Capture command output
* Check success/failure
* Save command output
* Log command execution

Examples:

* df -h
* free -m
* whoami
* uptime

---

### Script8. Server Resource Alert System

Continuously monitors server resources and alerts on high usage.

Features:

* CPU monitoring
* RAM monitoring
* Disk monitoring
* Warning alerts
* Warning count tracking
* Continuous monitoring
* CSV reporting

---

### Script9. JSON Config Reader + Deployment Validator

Validates deployment configuration files.

Features:

* Read JSON config files
* Validate required fields
* Detect invalid configuration
* Handle missing files
* JSON error handling
* Logging
* CSV reporting

Example:

```json
{
  "service": "nginx",
  "port": 80,
  "environment": "production"
}
```
